Intro to swmm-js

What are the advantages of working with swmm files as Javascript objects?

When developing applications for working with swmm files, there are plenty of options. Javascript just so happens to be an excellent choice for the following reasons:

  • No compiler
  • Free, open standard
  • Beautiful display options
  • Access to countless open-source libraries
  • Support across countless message boards
  • Can create interactive web pages
  • Can reside on a server or desktop
  • Incredibly easy data operations

This is a really good list, too. You can develop using a text editor and a modern web browser. You will not have to worry about paying any subscription or license fees for using Javascript. Javascript, when combined with HTML and CSS, is responsible for the world's most beautiful web pages. If you like a Javascript library, you only need to write the library address in your file. If you need help developing in Javascript, it is likely your question has been asked and answered a hundred times over. Developing in Javascript means you now have an interactive web page that you can show other people. Your code can remain private to you or you can share it with the world on a website. Finally, working with data in Javascript is satisfying and easy.
